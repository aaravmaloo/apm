## Prerequisites

- A terminal (Bash, Zsh, PowerShell, or CMD)
- **Go 1.21+** if building from source
- Windows, macOS, or Linux

## What You'll Accomplish

By the end of this section you will have:

1. Installed the `pm` binary on your system
2. Created an encrypted vault with a strong master password
3. Added, searched, and retrieved secrets
4. Understood the lock/unlock session model
5. Optionally set up cloud sync and autofill
